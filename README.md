This repository contains a collection of my Mobile Software Development labs completed during my third year of Computer Science at TU Dublin.

## Projects Overview

### Lab 1: Hello World App
A simple application that displays a personalized message. This project serves as an introduction to Android development, demonstrating basic UI elements and how to display text.

### Lab 2: Vertical Linear Layout
An example project showcasing the use of vertical linear layouts in Android. It features buttons arranged in a column, illustrating how to structure UI components effectively in a mobile application.

### Lab 3: UI Component Interactions
This lab explores UI components and their interactions using Kotlin:
1. **TextView Manipulation**: Created a layout with a TextView displaying "Hello World!" and manipulated its text programmatically.
2. **Button View**: Implemented a Button to count and display clicks, demonstrating event handling.
3. **EditText**: Added EditText components for user input to enable dynamic text interactions.
4. **RadioButton and Spinner**: Incorporated RadioButtons and a Spinner to enhance user selection options.
